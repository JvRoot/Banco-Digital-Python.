Banco Digital em Python.

Sobre o Projeto

O Banco Digital em Python é um projeto desenvolvido com o objetivo de colocar em prática conceitos fundamentais de programação utilizando a linguagem Python. O sistema simula operações bancárias básicas, permitindo o gerenciamento de usuários e contas bancárias de forma simples e intuitiva.

Este projeto foi desenvolvido para compor meu portfólio profissional em tecnologia, demonstrando conhecimentos em lógica de programação, estruturas condicionais, funções, manipulação de listas e dicionários em Python.

Funcionalidades.

Cadastro de usuários;

Criação de contas bancárias;

Depósitos;

Saques com limite de R$ 500,00 por operação;

Limite diário de até 3 saques por conta;

Transferências entre contas bancárias;

Consulta de saldo e extrato;

Listagem das contas cadastradas.

Tecnologias Utilizadas

Python 3
